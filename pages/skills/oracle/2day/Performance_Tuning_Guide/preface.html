2 Day + Performance Tuning Guide. Предисловие.

В предисловии объясняются следующие темы:

Для кого

Это руководство предназначено для администраторов БД Oracle (DBAs), которые хотят настроить и оптимизировать производительность БД Oracle. Прежде чем использовать этот документ, вы должны быть знакомы с администрированием БД Oracle.

В частности, данное руководство предназначено для следующих групп пользователей:

  • Администраторы баз данных Oracle, которые хотят приобрести навыки настройки производительности базы данных

  • Администраторы баз данных, которые являются новичками в БД Oracle

Доступность документации

Для получения информации о приверженности Oracle обеспечению доступности посетите веб-сайт программы Oracle Accessibility Program по адресу.

Сопутствующие документы

Для получения дополнительной информации о темах, затронутых в этом документе, см. Следующие документы::

  • Oracle Database Administrator’s Guide

  • Oracle Database Concepts

  • Oracle Database Performance Tuning Guide

  • Oracle Database SQL Tuning Guide

Во многих примерах в этой книге используются примеры схем, которые устанавливаются по умолчанию при выборе базового варианта установки с базой данных Oracle. Смотрите Oracle Database Sample Schemas для получения информации о том, как эти схемы были созданы и как вы можете их использовать.
