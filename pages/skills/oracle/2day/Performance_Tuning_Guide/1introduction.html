1. Введение

Как администратор БД Oracle (DBA), вы несете ответственность за производительность вашей БД Oracle. Настройка базы данных для достижения желаемого уровня производительности может оказаться непростой задачей, особенно для администраторов БД-новичков. Oracle Database 2 Day + Performance Tuning Guide – это краткое руководство, в котором описывается, как выполнять повседневные задачи по настройке производительности БД с использованием функций, предоставляемых Oracle Diagnostics Pack, Oracle Tuning Pack и Oracle Enterprise Manager Cloud Control (облачное управление).
