Руководство по настройке SQL. Предисловие.

В этом руководстве объясняется, как настроить Oracle SQL.

Для кого

Этот документ предназначен для администраторов БД и разработчиков приложений, которые выполняют следующие задачи:

  • Генерируют и интерпретируют планы выполнения SQL

  • Рапоряжаются статистикой оптимизатора

  • Воздействуют на оптимизатор с помощью параметров инициализации или подсказок SQL

  • Управляют совместным использованием курсора для инструкций SQL

  • Наблюдают за выполнением SQL

  • Выполняют трассировку приложений

  • Управляют средствами настройки SQL

  • Используют SQL Tuning Advisor или SQL Access Advisor

  • Управляют профилями SQL

  • Управляют базовыми значениями SQL

Сопутствующие документы

В этом руководстве предполагается, что вы знакомы с Oracle Database Concepts

Часто упоминаются следующие документы:

  • Oracle Database Data Warehousing Guide

  • Oracle Database VLDB and Partitioning Guide

  • Oracle Database SQL Language Reference

  • Oracle Database Reference

Во многих примерах в этой книге используются примеры схем, которые устанавливаются по умолчанию при выборе базового варианта установки с базой данных Oracle. Смотрите Oracle Database Sample Schemas для получения информации о том, как эти схемы были созданы и как вы можете их использовать.
